# Hero Images

Upload your case study hero image here.

## File to Upload:
- `hero-image.jpg` - Large hero image for case study header

## Specifications:
- **Size**: 1440x800px (16:9 aspect ratio)
- **Format**: JPG, PNG, or WebP
- **Quality**: High resolution, professional presentation
- **Content**: Project overview, key screens, or compelling visual

## Usage:
This image will be displayed:
- At the top of your case study page
- As the main visual introduction to your project
- In full width across the case study header

Choose an image that immediately showcases the transformation or key achievement of your AMIZONE redesign!
