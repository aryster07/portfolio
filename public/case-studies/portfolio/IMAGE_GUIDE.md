## üé® Screenshot Tips

### Desktop Screenshots (1920x1080 or higher)
1. Use clean browser (hide bookmarks, incognito mode)
2. Take full-page screenshots including fold areas
3. Show both light and dark themes
4. Capture hover states where relevant

### Mobile Screenshots (375x812 iPhone or 360x800 Android)
1. Use browser dev tools or actual device
2. Show navigation menu open/closed states
3. Capture touch interactions
4. Include status bar for realism

### Design Process Images
1. Export from Figma at 2x resolution
2. Use consistent artboard sizes
3. Include annotations where helpful
4. Show iteration progression
