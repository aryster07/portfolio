# Portfolio Case Study Images

## Image Organization

### Cover Photo
- **Location**: `/public/case-studies/portfolio/cover/`
- **File**: `portfolio-cover.png` or `portfolio-cover.jpg`
- **Purpose**: Main cover image for the portfolio case study

### Hero Section
- **Location**: `/public/case-studies/portfolio/hero/`
- **Files needed**:
  - `desktop-light.png` - Desktop screen in light mode
  - `desktop-dark.png` - Desktop screen in dark mode  
  - `mobile-light.png` - Mobile screen in light mode
  - `mobile-dark.png` - Mobile screen in dark mode

### Final Screens
- **Location**: `/public/case-studies/portfolio/final-screens/`
- **Files**: Any additional final implementation screenshots

## Image Specifications
- **Format**: PNG or JPG
- **Quality**: High resolution for desktop, optimized for web
- **Aspect Ratio**: Maintain original device proportions
- **File Size**: Optimized for web (under 2MB per image)

## Usage
These images will be automatically loaded and displayed in the portfolio case study component with proper responsive behavior and theme switching capabilities.
