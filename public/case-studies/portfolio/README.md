# Portfolio Website Case Study Assets

This folder contains all the visual assets for the portfolio website case study.

## Folder Structure

- `cover/` - Hero and cover images for the case study
- `old-designs/` - Previous portfolio versions and iterations
- `new-designs/` - Current portfolio design screens
- `wireframes/` - Information architecture, user flows, and wireframes
- `research/` - User research, competitor analysis, and design insights
- `prototypes/` - Interactive prototypes and component libraries
- `final/` - Final implementation screenshots and detailed views

## Image Guidelines

- Use high-resolution images (minimum 1920px wide for desktop screens)
- Maintain consistent naming conventions
- Include both light and dark theme variations where applicable
- Ensure images showcase the design process and final implementation
- Include mobile and desktop responsive versions
