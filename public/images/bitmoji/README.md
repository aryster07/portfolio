# Bitmoji Images

This folder contains bitmoji character images for the portfolio website.

Upload your bitmoji images here with the following naming convention:
- `designer-bitmoji.png` - For the UX/UI Designer character
- `photographer-bitmoji.png` - For the Photographer character
- `profile-bitmoji.png` - For the main profile bitmoji

Recommended image specifications:
- Format: PNG with transparent background
- Size: 200x200px minimum
- Quality: High resolution for retina displays
