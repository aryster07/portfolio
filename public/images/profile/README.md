# Profile Images

This folder contains profile pictures and related images for the portfolio website.

Upload your images here with the following naming convention:
- `profile-picture.jpg` - Main profile picture (circular)
- `hero-image.jpg` - Hero section background image
- `about-image.jpg` - About section profile image

Recommended image specifications:
- Format: JPG for photos, PNG for images with transparency
- Size: Minimum 1920x1080px for hero images, 500x500px for profile pictures
- Quality: Optimized for web (80-90% quality)
- Aspect ratio: 1:1 for profile pictures, 16:9 for hero images
